# Shift Left Coaching V1

A fast, responsive static website for Shift Left Coaching and Consultancy.

## Run locally

Open `index.html` directly, or run:

```bash
python -m http.server 3000
```

Then visit `http://localhost:3000`.

## Deploy

This site can be deployed directly to GitHub Pages, Vercel, Netlify, or Cloudflare Pages.

## V1 scope

- Responsive homepage
- Individual coaching
- Organizational coaching
- Shift Left method
- Discovery-call contact section
- Mobile navigation
- SEO title and description
